# Retrieve information from a conductivity calibration details file

`get_cond_details()` extracts a specific subset of the information
stored in a details file generated by HOBOware during
conductivity/salinity calibration.

## Usage

``` r
get_cond_details(path)
```

## Arguments

- path:

  The path to a details file created by HOBOware during conductivity
  calibration.

## Value

A nested list containing of a small subset of the information in `file`.

## See also

- [`get_do_details()`](https://umasscds.github.io/BuzzardsBay/reference/get_do_details.md)
  for a similar function to process a HOBOware details file generated
  while calibrating dissolved oxygen.

- [`parse_hoboware_details()`](https://umasscds.github.io/BuzzardsBay/reference/parse_hoboware_details.md)
  for a function to read and format all of the information in a HOBOware
  details file.

## Examples

``` r
 file <- system.file(
   "extdata/2023/RB1/2023-06-09/Calibrated/Cond_RB1_2023-06-09_Details.txt",
                     package = "BuzzardsBay")
 do_cal  <- get_cond_details(file)
 cat(yaml::as.yaml(do_cal))
```
