# Import data from MX801 logger

Import data from MX801 logger

## Usage

``` r
read_mx801_data(file)
```

## Arguments

- file:

  Path to a calibrated Excel (`.xlsx`) file from an MX801 logger Column
  descriptions in [google
  doc](https://docs.google.com/spreadsheets/d/1bYxi0nbgDaUsKEyLoj_ry-1Zc-A4MNDfS_J_zH7HiJI)

## Value

A data frame with standard data columns
